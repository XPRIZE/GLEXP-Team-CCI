Web console build instructions

Spin up a Ubuntu 16.04 LTS server
Install apache2
Install php5
Install mysql

Move html and includes into the web directory of server (/var/www/)
Create new database named "console"
Import "console.sql" as db structure
Create mysql user named "console" with root access to "console" db
Open "includes/dbConnect.php" and change line 21 from 
	$this->PASSWORD = "RootPasswordHere";
to whatever password you set for user "console" on db "console"
Server is now ready to use.
