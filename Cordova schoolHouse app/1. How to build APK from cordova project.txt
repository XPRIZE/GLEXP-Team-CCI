For plugin https://github.com/oddmouse/cordova-plugin-locktask, follow his github instructions for how to create an admin reciever java class. Name said class "MyAdmin"

Put schoolhouse contents into cordova web dir. Either

	Change the index files over based on index templates used for cordova app

OR 

	Download cordova school house application www guts from one of the three official builds online

To build extremely large APKs from cordova, do the following.

First, clean.

> cordova clean android

Then, build from powershell

> cordova build android --verbose

It WILL fail, after about 30 minutes per gb

Then, build again

> cordova build android --verbose

Keep building from powershell until the failure takes >3 minutes.

Then, from a cmd.exe as admin, cd to the project, increase java heap size

Increase java memory to 2X the size of the www folder in the project

> SET _JAVA_OPTIONS="-Xmx4000m"

Build again. 

> cordova build android --verbose

It will also probably fail

Then, from task manager, kill the java process with all the ram in it. Build a final fime

> cordova build android --verbose

It will work, but from cmd.exe as admin only.
